# Expense-Tracker
This is an simple mobile application using Flutter , " Expense Tracker " Where the user can able to add the expense they have and delete if they want ... Very simple , contain simple detail like Name of the expense , Date and the Amount .
with the package included [intl , hive_flutter , build_runner , hive_generator]
Finally to use this app as an apk file 
flutter build apk --release to run in our physical device through apk
